关于 MyDog  
Introduce

MyDog 是一个自动代码生成工具。
[MyDog 开源地址]https://github.com/PowerShenli/MyDog

最初版支持一键生成一个基于Spring Boot的web工程。包括项目骨架和实体的CRUD功能以及web界面。

有别于其他代码生成器的特点

由于所有功能均是通过插件实现的. 所以理论上讲,只要扩展更多的插件,功能的扩展也将是无限的.
